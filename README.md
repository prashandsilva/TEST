## 🚀 Quick Start

### 1. Run the Python Calculator
```bash
python3 calculator.py
```

### 2. Run Tests
```bash
python3 test_calculator.py
```

### 3. View Web Calculator
Open `index.html` in your web browser to see the interactive calculator.

## 🔧 Tools You Can Use

### Text Editors
- **VS Code** (recommended for beginners)
- **Vim/Neovim** (advanced users)
- **Sublime Text**
- **Atom**

### Version Control
```bash
git status          # Check what files changed
git diff            # See exact changes
git add .           # Stage changes
git commit -m "..."  # Commit with message
```

### Linting (Optional)
```bash
# Python
pip install pylint
pylint calculator.py

# JavaScript
npm install -g eslint
eslint script.js
```

## 💡 Key Principles

1. **Understand before editing** - Read and comprehend existing code
2. **Make small changes** - Edit incrementally and test frequently
3. **Follow patterns** - Maintain consistency with existing code style
4. **Test everything** - Verify changes work as expected
5. **Document changes** - Add comments and update documentation
6. **Handle errors** - Consider edge cases and failure scenarios
